# 🌊 Karimunjawa Interaktif

Project eksplorasi wisata Karimunjawa berbasis **React**. Dibuat untuk memudahkan wisatawan menjelajahi keindahan pulau secara digital.

## 🚀 Rencana Fitur

- [x] **Responsive Design**: Nyaman dibuka di HP maupun Laptop. (Sudah kita kerjain barusan!)
- [ ] **Interactive Map**: Peta spot wisata menarik di Karimunjawa.
- [ ] **Gallery**: Koleksi foto keindahan alam bawah laut.
- [ ] **Tour Guide**: Informasi paket wisata dan penginapan.

## 🛠️ Tech Stack

- **Framework**: React.js
- **Styling**: CSS Modern / Tailwind CSS (Rencana)
- **Deployment**: GitHub Pages
