# 🌊 Karimunjawa Interaktif

Project eksplorasi wisata Karimunjawa berbasis **React**. Dibuat untuk memudahkan wisatawan menjelajahi keindahan pulau secara digital.

## 🚀 Rencana Fitur

- [x] **Interactive Map**: Peta spot wisata menarik di Karimunjawa.
- [x] **Gallery**: Koleksi foto keindahan alam bawah laut.
- [x] **Tour Guide**: Informasi paket wisata dan penginapan.
- [] **Responsive Design**: Nyaman dibuka di HP maupun Laptop.

## 🛠️ Tech Stack

- **Framework**: React.js
- **Styling**: CSS Modern / Tailwind CSS (Rencana)
- **Deployment**: GitHub Pages
